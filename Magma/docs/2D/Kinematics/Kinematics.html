StartSolver

StartSolver (world: ^World)

starts the kinematics solver in a new thread

Parameters:
  • world: a pointer to the world to create a solver for

StopSolver

StopSolver (world: ^World)

stops the kinematics solver associated with the world

Parameters:
  • world: a pointer to the world that owns the solver we want to stop

MoveObject

MoveObject (world: ^World, id: ObjectID, amount: Types.Vector2f)

attempt to move an object in a world by a certain amount note that this acts more like a maximum posible as the kinematics will check for collisions and could cause the object to not move at all

Parameters:
  • world: a pointer to the world that owns the object we want to move
  • id: the id of the object we want to move
  • amount: the amount in pixels we want to move

IsCollidingWith

IsCollidingWith (world: ^World, id: ObjectID)

checks if an object is coliding with another object and gives you the id of the other object

Parameters:
  • world: a pointer to the world that owns the object to check and the returned id
  • id: the id of the object we want to check
Returns: the id of the object that we collided with and true if we collided with anything or false if not

GetCollisions

GetCollisions (world: ^World, id: ObjectID)

similar to IsCollidingWith but returns multiple objects that the object is colliding with and in complex scenes is prefered over it

Parameters:
  • world: a pointer to the world that owns the objects we want to check
  • id: the id of the object we want to check collisions against
Returns: a list of collisions and their info
